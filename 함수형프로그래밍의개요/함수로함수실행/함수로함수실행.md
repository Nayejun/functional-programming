## 함수로 함수 실행하기
```
function f4(f1, f2, f3) {
    return f3(f1() + f2());
    // f1() = 2, f2() = 1
    // 2 + 1 = 3
}

f4 (
    function() { return 2; }
    function() { return 1; }
    function(a) { return a * a; }
);
```
> 위의 코드에서 각각의 함수들은 모두 **순수함수**인데,<br /> 이것들을 **조합하여 순수함수들이 결과**를 만들면 출력을 하고,<br /> 그것들을 조합하는 ```f4``` 같은 함수를 통해
함수들의 **평가 시점**이나, **평가 방법**이나,<br /> 어떤 로직에서 평가를 할지를 결정하는 **큰 로직을 만들어가는 프로그래밍 방법이다.**
