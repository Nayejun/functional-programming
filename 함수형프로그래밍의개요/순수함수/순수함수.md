## 순수함수란 ?
1. 항상 동일한 인자를 주면 동일한 결과를 return 하는 함수

2. 부수효과가 없는 함수<br />
-> 부수효과 : 외부의 상태에 영향을 미치는 것

```
function add(a, b) {
    return a+b;
}
console.log( add(10, 5));
console.log( add(10, 5));
console.log( add(10, 5));
```
