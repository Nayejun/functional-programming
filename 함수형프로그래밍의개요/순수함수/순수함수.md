**2. 부수효과를 일으키는 함수**
<br><br>
***부수효과***<br>
-> 외부의 상태를 변경하는 것.<br>
->들어온 인자의 상태를 직접 변경하는 것.

```
var c = 20
function add3(a,b) {
    c = b;
    return a + b;
}
```
